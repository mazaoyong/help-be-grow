### 使用说明
#### API
name | 类型 | 说明 | 默认值
-|-|-
id | number或string | 音频唯一id，为了兼容多个音频相同的情况 | -
audioUrl | string  | 音频 | -
mediaName | string | 音频名称 | -
width | string | 控件宽度 | 364px
height | string | 控件高度 | 68px
