# 推广组件

```tsx
<ActivityPromotion alias={data.alias} title="转介绍活动">推广</ActivityPromotion>
```

- `alias: string` 活动alias
- `title: string` 活动名

制作推广海报，在推广海报页面点击返回时回到活动列表页
