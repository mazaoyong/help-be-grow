# 督学 - 群打卡

## 包含的页面

- introduction：打卡介绍页
- edit: 打卡编辑页
- task: 打卡任务详情页
- detail: 打卡详情页，作为分享的落地页
- list: 打卡区，打卡列表页
- rank: 排行榜/我的打卡

## 外部入口

- 微页面
- 专栏内容页
- 我的课程 - 打卡
- 订单列表/详情
- 分享、日签、长图
- 模板消息

## 支持访问环境

- mobile
  - 微信浏览器
  - 非微信浏览器
- 百度小程序

## 支持店铺类型

- 微商城单店
- 微商城连锁
- 教育单店
- 教育连锁校区
